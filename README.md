## ğŸ“– **Project Description (English)**

Our game is inspired by Wordle spin-offs like [Gamedle](https://www.gamedle.wtf/) or [PokÃ©dle](https://pokedle.net/). Taking advantage of the upcoming Eurovision Song Contest, we decided to base the game on this event.
